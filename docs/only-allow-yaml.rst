Only allow YAML
---------------




.. code-block:: python

    >>> from strictyaml import Str, Int, load

.. code-block:: python

    >>> load(None, Str())
    EXCEPTION RAISED:
      StrictYAML can only read a string of valid YAML.



.. code-block:: python

    >>> load("1", Str()) == "1"
    True



.. code-block:: python

    >>> load("1", Int()) == 1
    True



.. code-block:: python

    >>> load("", Str()) == ""
    True



.. code-block:: python

    >>> load("x:") == {"x": ""}
    True



.. code-block:: python

    >>> load("null: null") == {"null": "null"}
    True
